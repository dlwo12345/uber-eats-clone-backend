## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
https://nomadcoders.co/nuber-eats/ 노마드코더 우버이츠 클론 코딩 study

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## 추가 패키지

- graphql 패키지
> npm i @nestjs/graphql graphql-tools graphql apollo-server-express

- class-validator
> npm i class-validator
